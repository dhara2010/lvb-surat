# LVB Surat - Architecture restructuring

A full-stack monorepo web application for LVB Surat.

## Folder Structure

- `frontend/`: React + Vite frontend for regular users.
- `backend/`: Node.js + Express backend providing APIs.
- `admin/`: React + Vite admin dashboard (to be fully implemented).
- `docs/`: Project documentation.
- `database/`: Database notes and schema setups.
- `shared/`: Shared enums, types, or validation schemas.

## Technologies Used

- **Frontend:** React, Vite, Tailwind CSS, Framer Motion
- **Backend:** Node.js, Express, MongoDB (planned integration)
- **Admin:** React, Vite

## Getting Started

To run the full stack (once dependencies are installed):

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

### Admin
1. `cd admin`
2. `npm install`
3. `npm run dev`
